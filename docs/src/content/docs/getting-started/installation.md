## Homebrew (macOS / Linux)

```bash
brew install typemd
```

## From Source

Make sure you have [Go](https://go.dev/) installed, then:

```bash
go install github.com/typemd/typemd@latest
```

## Verify Installation

```bash
tmd --version
```
